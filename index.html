Process Documentation

Methodology

A transparent account of the analytical frameworks, software workflows, and quality control measures applied across all projects in this portfolio.

Core Framework

Analytical Approach

Step 01
Data Acquisition
All spatial datasets are sourced from reputable repositories including USGS Earth Explorer, Copernicus Open Access Hub, and national geospatial agencies. Data quality is assessed prior to any processing by reviewing metadata, coverage, and resolution specifications.
USGS EarthExplorer Copernicus Hub OpenStreetMap Google Earth Engine
Step 02
Pre-Processing & Cleaning
Raw data undergoes geometric correction, atmospheric correction (for satellite imagery), and coordinate system transformation to a consistent projection. Outliers and null values are identified and handled systematically before analysis.
QGIS ArcGIS Pro ENVI Python (GDAL)
Step 03
Spatial Analysis
Analytical methods are chosen according to the research question — from raster-based overlay analysis and vector geoprocessing, to more advanced techniques such as suitability modelling, change detection, and network analysis.
Raster Analysis Vector Geoprocessing ModelBuilder R (sf package)
Step 04
Cartographic Design
Maps are designed following established cartographic principles: appropriate projection selection, thoughtful colour ramp choices, clear legend hierarchies, and inclusion of all essential map elements. Visual clarity is prioritised throughout.
QGIS Print Layouts ArcGIS Layouts Adobe Illustrator ColorBrewer
Step 05
Validation & Accuracy Assessment
Where applicable, results are validated through ground truth sampling, confusion matrices (for classified outputs), and cross-referencing against secondary data sources. Accuracy metrics are documented and reported.
Confusion Matrix Kappa Coefficient Field Verification Statistical Testing
Step 06
Documentation & Reporting
All projects are fully documented with methodology notes, data citations, software versions, and analytical decision logs. Final outputs are stored in standardised folder structures to ensure reproducibility.
Metadata Standards ISO 19115 LaTeX / Word Version Control
